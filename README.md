>deep learning - 不需要很深的數學，不推導kernel背後數學演譯邏輯，僅用"猜答案"的訓練手法，"算出"最合適的係數or找到最好的解法(其實過程就是把圖畫出來，按照現有模式寫程式，就能自己架net，電腦自動幫你跑)
